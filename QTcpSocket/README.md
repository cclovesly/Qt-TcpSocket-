# 简单实现Tcp服务器端和客户端通讯

客户端实现了类似QQ登录界面，通过数据库查询，判断账号以及密码的正确性，通过添加数据库数据实现注册账号密码的功能，聊天大厅界面，使用随机生成来实现好友的数量，点击好友进入客户端通讯界面，就可以与服务器端进行聊天通讯了。
